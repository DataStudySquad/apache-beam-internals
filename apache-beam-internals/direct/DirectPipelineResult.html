DirectPipelineResult

DirectPipelineResult is a PipelineResult of DirectRunner.

Creating Instance

DirectPipelineResult takes the following to be created:

  • PipelineExecutor

  • EvaluationContext

DirectPipelineResult is created when DirectRunner is requested to run a pipeline.
